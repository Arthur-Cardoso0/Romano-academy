Academia Romano — Site Oficial
📌 Sobre o projeto
Este repositório contém o site oficial da Academia Romano, desenvolvido para apresentar planos, permitir pré‑matrícula online e oferecer um formulário de contato integrado.

🎨 Design inspirado nas cores vermelho e preto, transmitindo força e disciplina.

💻 Código criado com apoio de Inteligência Artificial (IA) para acelerar o desenvolvimento e estruturar funcionalidades modernas.

📩 Integração com EmailJS para envio de pré‑matrículas e mensagens de contato diretamente para a equipe.

🔒 Projeto autoral, criado exclusivamente para a Academia Romano.

🚀 Funcionalidades
Modal de pré‑matrícula com validação de dados.

Formulário de contato com envio direto para o e‑mail da academia.

Layout responsivo e otimizado para diferentes dispositivos.

Estrutura simples e clara para fácil manutenção.

🛠️ Tecnologias utilizadas
HTML5 / CSS3 / JavaScript

EmailJS para envio de mensagens sem backend próprio

Design responsivo com foco em experiência do usuário

📩 Como funciona o envio de mensagens
O cliente preenche o formulário de pré‑matrícula ou contato.

Os dados são enviados via EmailJS para o e‑mail fixo da Academia Romano.

O cliente não recebe cópia — apenas a equipe Romano é notificada.
